TaroPet For Windows
=============

### Environment ###
- "nodemon": "^2.0.22"
- "electron": "24.3.1"
- "electron-store": "^8.1.0",
- "@electron-forge/cli": "^6.2.1",

### How to build ###

install

```sh
pnpm install
```

Then, to start

```sh
pnpm start
```

package

```sh
pnpm run make
```
