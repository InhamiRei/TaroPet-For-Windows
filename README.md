### Environment ###
- "nodemon": "^2.0.22"
- "electron": "24.3.1"
- "electron-store": "^8.1.0",
- "@electron-forge/cli": "^6.2.1",

### How to build ###

```sh
pnpm install
```

Then, to build the source, run:

```sh
pnpm start
```
